Projeto Streaming

Este projeto de streaming foi desenvolvido utilizando HTML e CSS para criar uma interface visual atraente e responsiva. Embora a maior parte da estrutura e do estilo tenha sido implementada com essas tecnologias, o JavaScript foi utilizado de forma pontual para adicionar interatividade básica, no botão de "Categorias" e na lupa de pesquisa.

Projeto Login

Este projeto é uma página de login simples, desenvolvida com HTML e CSS. A interface permite que os usuários insiram suas credenciais de forma intuitiva, com campos para nome de usuário e senha. O design é limpo e responsivo, garantindo uma boa experiência em diferentes dispositivos.
